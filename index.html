🔀 ESTRUCTURA SELECTIVA

Estructuras de control que permiten tomar decisiones en el programa mediante condiciones. Incluye if, if-else, else-if, switch-case y operador ternario para controlar el flujo de ejecucion.

📁 Trabajos realizados

  1. 📊 EJEMPLO_CARRO
  2. 🎯 EJEMPLO_AREAS
  3. 🔢 EJEMPLO DE LADRILLOS

🔄 ESTRUCTURA REPETITIVA

Ciclos o bucles que permiten ejecutar un bloque de codigo multiples veces. Incluye for, while, do-while y tecnicas de iteracion para procesamiento eficiente de datos.

📁 Trabajos realizados

  1. ✖ Generador de Tablas de Multiplicar
  2. 🔢 Calculador de Numeros Primos
  3. 🌀 Serie de Fibonacci

📦 ENCAPSULAMIENTO

Principio de POO que oculta los detalles internos de una clase y proporciona una interfaz publica. Incluye modificadores de acceso (private, public, protected), getters, setters y validacion de datos.

📁 Trabajos realizados

  1. 🎓 Clase Estudiante Encapsulada
  2. 🏦 Sistema Bancario con Encapsulamiento
  3. 📱 Clase Producto con Validaciones

⚙ METODOS Y SOBRECARGAS

Funciones dentro de clases que definen el comportamiento de los objetos. La sobrecarga permite crear multiples metodos con el mismo nombre pero diferentes parametros (polimorfismo en tiempo de compilacion).

📁 Trabajos realizados

  1. 🧮 Calculadora con Metodos Sobrecargados
  2. 📐 Clase Figura con Sobrecarga
  3. 💰 Sistema de Pagos Sobrecargado

🏗 CONSTRUCTORES Y DESTRUCTORES

Metodos especiales para inicializar y destruir objetos. Los constructores se ejecutan al crear un objeto, mientras que los destructores liberan recursos al destruirlo. Incluye constructores por defecto, parametrizados y de copia.

📁 Trabajos realizados

  1. 🚗 Clase Vehiculo con Constructores
  2. 📚 Clase Libro con Constructor de Copia
  3. 🏠 Gestor de Memoria con Destructores

🔗 RELACIONES ENTRE CLASES

Interacciones entre diferentes clases en POO. Incluye asociacion, agregacion, composicion y herencia. Permite modelar relaciones del mundo real: tiene un, es un, usa, contiene, etc.

📁 Trabajos realizados

  1. 🏫 Sistema Universidad (Asociacion)
  2. 🚙 Auto y Motor (Composicion)
  3. 👥 Sistema de Empleados (Herencia)
